# Brainstorms

Design explorations and their outcomes, preserved so decisions (and the reasoning behind them) survive.

## Convention

- **Open sessions live directly in this directory** (`docs/brainstorm/*.md`) — designs still under discussion or not yet implemented.
- **Finished ones move to `archived/`** once the decision has shipped, so this directory shows only what's still live.

Each doc carries a **Status** line at the top (`OPEN` / `DONE`) and, when done, the commit(s) that implemented it.
